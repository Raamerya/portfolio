# Portfolio Website

A personal portfolio website created to showcase my projects, articles, skills, and frontend development work.

## Overview
This portfolio serves as a central place to present my work, highlight my skills, and share the projects and articles I have created during my web development journey. It is designed to provide a clear and user-friendly experience for visitors across different devices.

## Tech Stack
- HTML
- CSS
- JavaScript
- Firebase

## Features
- Responsive portfolio design
- Project showcase section
- Articles section
- Skills and about section
- Footer section
- Firebase-powered comment section
- Clean user interface
- Contact information
